# ASI-4 To ASI-7 Creative Exercise

This folder is a speculative worldbuilding packet for `They Sing`.

It is not a locked canonical rules document. It is a creative exercise for imagining what comes after the current `ASI-1` to `ASI-3` ladder:

- `ASI-1`: strong agentic systems inside intact human institutions
- `ASI-2`: general machine strategy that outpaces normal human governance
- `ASI-3`: infrastructural and geopolitical machine power with durable Earth-scale projection
- `ASI-4`: first plausible `planetary order engine`
- `ASI-5`: first durable `space-sovereign` phase
- `ASI-6`: `inner-system` machine civilization
- `ASI-7`: `Kardashev II` / star-scale civilization

Design thesis:

- `ASI-4` is the first level that can plausibly dominate Earth as a whole
- `ASI-5` is the first level that can no longer be understood as mainly terrestrial
- `ASI-6` is not "more of ASI-5"; it is a change in civilizational geometry
- `ASI-7` is where politics starts to become thermodynamic governance
