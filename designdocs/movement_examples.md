# Movement Examples

This is a seed book for the movement system in [movements.md](./movements.md).

These are not meant to be a fixed ideology roster.
They are example outputs that show the intended range of generative memetics.

Each example includes:

- tonal shape
- recruitment signature
- likely short-run effect
- likely long-run risk

## 1. Municipal Competence Front

- grievance: corruption
- promise: reform
- authority: expert
- epistemic: empirical
- social form: policy caucus
- sacrifice: civic duty
- AI relation: adviser
- aesthetic: boring competence

Recruitment:

- many `rubes`
- moderate `contractors`
- slower `trueBelievers`

Effect:

- raises TAS tolerance through plausibility
- spreads by making mocked ideas work locally

Risk:

- can hollow into procedural capture

## Design Read

The intended range is:

- some movements are obviously alarming
- some look sane until very late
- some are mostly social
- some are mostly organizational
- some are mostly sacred
- some are mostly contractual

The generator should be able to produce all of these without needing 20 hard-coded factions.
